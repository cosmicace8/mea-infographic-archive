タイトル

A Dual-Band RF Exposure System for Real-Time Electrophysiological Recordings Using Microelectrode Array Device

著者

N Dolciotti, C Pisano, A Paffi, M Cioni, G Guerra, I Zironi…

日本語要約

本論文は、マイクロ電極アレイ（MEA）デバイスを用いたリアルタイム電気生理学的記録のためのデュアルバンドRF曝露システムについて記述しています。特に、神経細胞培養におけるRF曝露の影響を評価するために設計されており、デュアルバンド対応が特徴です。
